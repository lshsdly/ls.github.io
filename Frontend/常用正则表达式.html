          # 数字校验

          • 数字
            • ^[0-9]*$
          • n位的数字
            • ^\d{n}$
          • 至少n位的数字
            • ^\d{n,}$
          • m-n位的数字
            • ^\d{m,n}$
          • 零和非零开头的数字
            • ^(0|[1-9][0-9]*)$
          • 非零开头的最多带两位小数的数字
            • ^([1-9][0-9]*)+(.[0-9]{1,2})?$
          • 带1-2位小数的正数或负数
            • ^(-)?\d+(.\d{1,2})?$
          • 正数、负数、和小数
            • ^(-|+)?\d+(.\d+)?$
          • 有两位小数的正实数
            • ^[0-9]+(.[0-9]{2})?$
          • 有1~3位小数的正实数
            • ^[0-9]+(.[0-9]{1,3})?$
          • 整数 　* ^-?[1-9]\d*$
          • 非零的正整数
            • ^[1-9]\d*$
            • ^([1-9][0-9]*){1,3}$
            • ^+?[1-9][0-9]*$
          • 非零的负整数
            • ^-[1-9][]0-9"*$
            • ^-[1-9]\d*$
          • 非负整数
            • ^\d+$
            • ^[1-9]\d*|0$
          • 非正整数
            • ^-[1-9]\d*|0$
            • ^((-\d+)|(0+))$
          • 浮点数
            • ^(-?\d+)(.\d+)?$
            • ^-?([1-9]\d*.\d*|0.\d*[1-9]\d*|0?.0+|0)$
          • 正浮点数
            • ^[1-9]\d*.\d*|0.\d*[1-9]\d*$
            • ^(([0-9]+.[0-9][1-9][0-9])|([0-9][1-9][0-9].[0-9]+)|([0-9][1-9][0-9]))$
          • 负浮点数
            • ^-([1-9]\d*.\d*|0.\d*[1-9]\d*)$
            • ^(-(([0-9]+.[0-9][1-9][0-9])|([0-9][1-9][0-9].[0-9]+)|([0-9][1-9][0-9])))$
          • 非负浮点数
            • ^\d+(.\d+)?$
            • ^[1-9]\d*.\d*|0.\d*[1-9]\d*|0?.0+|0$
          • 非正浮点数
            • ^((-\d+(.\d+)?)|(0+(.0+)?))$
            • ^(-([1-9]\d*.\d*|0.\d*[1-9]\d*))|0?.0+|0$

          # 字符校验

          • 汉字
            • ^[\u4e00-\u9fa5]{0,}$
          • 英文和数字
            • ^[A-Za-z0-9]+$
            • ^[A-Za-z0-9]{4,40}$
          • 长度为3-20的所有字符
            • ^.{3,20}$
          • 由26个英文字母组成的字符串
            • ^[A-Za-z]+$
          • 由26个大写英文字母组成的字符串
            • ^[A-Z]+$
          • 由26个小写英文字母组成的字符串
            • ^[a-z]+$
          • 由数字和26个英文字母组成的字符串
            • ^[A-Za-z0-9]+$
          • 由数字、26个英文字母或者下划线组成的字符串
            • ^\w+$
            • ^\w{3,20}$
          • 中文、英文、数字包括下划线
            • ^[\u4E00-\u9FA5A-Za-z0-9_]+$
          • 中文、英文、数字但不包括下划线等符号
            • ^[\u4E00-\u9FA5A-Za-z0-9]+$
            • ^[\u4E00-\u9FA5A-Za-z0-9]{2,20}$
          • 可以输入含有^%&',;=?$"等字符
            • [^%&',;=?$\x22]+
          • 禁止输入含有~的字符
            • [^~\x22]+
          • 不以 xxx (如 jeffjade)开头/结尾的字符串
            • ^(?!jeffjade).*$
            • ^.*?(?<!jeffjade)$

          # 特殊表达式校验

          • 中文字符
            • [\u4e00-\u9fa5]
          • 双字节字符(含汉字)
            • [^\x00-\xff] 　* 可以用来计算字符串的长度（一个双字节字符长度计 2，ASCII 字符计 1）
          • 空白行
            • \n\s*\r 　* 可以用来删除空白行
          • 首尾空白字符
            • ^\s*|\s*$
            • (^\s*)|(\s*$) 　* 可以用来删除行首行尾的空白字符，含空格、制表符、换页符等
          • 网址 URL
            • [a-zA-z]+://[^\s]*
            • [-a-zA-Z0-9@:%.\+~#=]{2,256}.[a-z]{2,6}\b([-a-zA-Z0-9@:%\+.~#?&//=]*)
              • 不必包含http or https
            • ^http://([\w-]+.)+[\w-]+(/[\w-./?%&=]*)?$
            • https?😕/(www.)?[-a-zA-Z0-9@:%.\+~#=]{2,256}.[a-z]{2,6}\b([-a-zA-Z0-9@:%\+.~#?&//=]*)
              • 必须包含http / https
          • Email 地址
            • ^[a-zA-Z0-9_-]+@[a-zA-Z0-9_-]+(.[a-zA-Z0-9_-]+)+$
            • ^([a-z0-9_\.-]+)@([\da-z\.-]+)\.([a-z\.]{2,6})$
            • ^\w+([-+.]\w+)@\w+([-.]\w+).\w+([-.]\w+)*$
            • \w[-\w.+]*@([A-Za-z0-9][-A-Za-z0-9]+.)+[A-Za-z]{2,14}
          • 域名
            • [a-zA-Z0-9][-a-zA-Z0-9]{0,62}(/.[a-zA-Z0-9][-a-zA-Z0-9]{0,62})+/.?
          • ip地址
            • \d+.\d+.\d+.\d+
          • HTML 标记
            • <(\S*?)[^>]>.?</\1>|<.*? /> 　* 网上流传的版本太糟糕，上面这个也仅能匹配部分，对于复杂的嵌套标记依旧无能为力
          • xml 文件
            • ^([a-zA-Z]+-?)+[a-zA-Z0-9]+\.[x|X][m|M][l|L]$
          • 合法 img 标签
            • <img.\ssrc=.?>

          # 日期

          • 日期格式
            • ^\d{4}-\d{1,2}-\d{1,2}
          • 一年的 12 个月(01～09 和 1～12)
            • ^(0?[1-9]|1[0-2])$
          • 一个月的 31 天(01～09 和 1～31)
            • ^((0?[1-9])|((1|2)[0-9])|30|31)$
          • YYYY-MM-DD
            • ^((((1[6-9]|[2-9]\d)\d{2})-(0?[13578]|1[02])-(0?[1-9]|[12]\d|3[01]))|(((1[6-9]|[2-9]\d)\d{2})-(0?[13456789]|1[012])-(0?[1-9]|[12]\d|30))|(((1[6-9]|[2-9]\d)\d{2})-0?2-(0?[1-9]|1\d|2[0-8]))|(((1[6-9]|[2-9]\d)(0[48]|[2468][048]|[13579][26])|((16|[2468][048]|[3579][26])00))-0?2-29-))$
            • 基本上把闰年和2月等的情况都考虑进去了
            • 加时间验证
              • ^((((1[6-9]|[2-9]\d)\d{2})-(0?[13578]|1[02])-(0?[1-9]|[12]\d|3[01]))|(((1[6-9]|[2-9]\d)\d{2})-(0?[13456789]|1[012])-(0?[1-9]|[12]\d|30))|(((1[6-9]|[2-9]\d)\d{2})-0?2-(0?[1-9]|1\d|2[0-8]))|(((1[6-9]|[2-9]\d)(0[48]|[2468][048]|[13579][26])|((16|[2468][048]|[3579][26])00))-0?2-29-)) (20|21|22|23|[0-1]?\d):[0-5]?\d:[0-5]?\d$

          # money

          • ^(0|[1-9][0-9]*)$
            • 同非负整数
            • 可以允许开头有一个负号：^(0|-?[1-9][0-9]*)$
          • 可能的小数部分
            • ^[0-9]+(.[0-9]+)?$
              • 小数点后面至少应该有1位数,所以"10."是不通过的,但是 "10" 和 "10.2" 是通过的
            • ^[0-9]+(.[0-9]{2})?$
              • 这样规定小数点后面必须有两位
            • ^[0-9]+(.[0-9]{1,2})?$
              • 这样允许用户只写一位小数
          • 逗号分隔(如 3,250.00)
            • ^[0-9]{1,3}(,[0-9]{3})*(.[0-9]{1,2})?$
              • 1-3 个数字,后面跟着任意个逗号+3个数字
            • ^([0-9]+|[0-9]{1,3}(,[0-9]{3})*)(.[0-9]{1,2})?$
              • 逗号可选而不是必须
          • 货币格式
            • '123123211312.333333'.replace(/(?=(?!^)(?:\d{3})+(?:.|$))(\d{3}(.\d+$)?)/g, ',$1') //输出 123,123,211,312.333333
            • ps：TCL 大牛推荐

          # 表单常用

          • 帐号是否合法(字母开头，5-16 字节，允许字母数字下划线)
            • ^[a-zA-Z][a-zA-Z0-9_]{4,15}$
          • 手机号
            • ^(13[0-9]|14[5|7]|15[0|1|2|3|5|6|7|8|9]|18[0|1|2|3|5|6|7|8|9])\d{8}$
          • 国内电话号码
            • ^((\d{3,4}-)|\d{3.4}-)?\d{7,8}$
              • XXX-XXXXXXX / XXXX-XXXXXXXX / XXX-XXXXXXX / XXX-XXXXXXXX / XXXXXXX / XXXXXXXX 　* \d{3}-\d{8}|\d{4}-\d{7}
              • XXXX-XXXXXXX / XXX-XXXXXXXX
          • 身份证号
          • ^\d{15}|\d{18}$
            • 15 / 18 位
          • ^[1-9]\d{7}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{3}$
            • 15 位
          • ^[1-9]\d{5}[1-9]\d{3}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{4}$
            • 18 位
          • ^([0-9]){7,18}(x|X)?$ 或 ^\d{8,18}|[0-9x]{8,18}|[0-9X]{8,18}?$
            • 短身份证号码(数字、字母x结尾)
          • 密码(字母开头，长度在 6~18 之间，只能包含字母、数字和下划线)
            • ^[a-zA-Z]\w{5,17}$
          • 强密码(必须包含大小写字母和数字的组合，不能使用特殊字符，长度在 8-10 之间)
            • ^(?=.\d)(?=.[a-z])(?=.*[A-Z]).{8,10}$
          • QQ 号
            • ^[1-9][0-9]{4,}$ 　* QQ 号从 10000 开始
          • 微信号
            • ^[A-Za-z][\w-]{5,19}$;
            • 6-20 个字母、数字、下划线和减号，必须以字母开头(不区分大小写)
          • 中国邮编
            • [1-9]\d{5}(?!\d) 　* 中国邮政编码为6位数字
